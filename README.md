## アプリ名
　Sports_Park

## サイト概要
　スポーツに参加したい人向けの募集サイト

## サイトテーマ
　スポーツ募集サイト

## テーマを選んだ理由
　Labolaというサイトを参考に作成しました。
  実際にサイトを利用してみて、地図機能や天気の機能があればより利用しやすくなると思い作成しようと思いました。

## ターゲットユーザ
　スポーツを行う人

## 主な利用シーン
　スポーツに参加したい時
